# Fake News Detector for Students

## Problem Statement
Misinformation spreads rapidly through online news and social media.
Students find it difficult to differentiate between real and fake news.

## Proposed Solution
An AI-based system that analyzes news articles using
machine learning and NLP techniques to classify them
as Fake or Real.

## Technologies Used
- Python
- Machine Learning
- Natural Language Processing (NLP)
- Streamlit

## System Workflow
1. User enters news article text
2. Text preprocessing is done
3. Machine learning model analyzes content
4. Output is displayed as Fake or Real

## Deployment
- Streamlit Web Application
- Runs in a web browser

## Future Scope
- Multi-language support
- Browser extension
- Deep learning models
